### 三、功能描述
系统分为（身份）：普通用户、教师、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 班级信息
6. 请假表格
7. 提交请假表
8. 学生考勤
9. 缺课记录


**教师功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人中心
5. 学生信息
6. 班级信息
7. 请假表格管理
8. 提交请假表格管理
9. 学生考勤管理
10. 缺课记录管理



**管理员功能：**
1. 登录
2. 个人中心
3. 学生管理
4. 教师管理
5. 班级信息管理
6. 请假表格管理
7. 提交请假表管理
8. 学生考勤管理
9. 缺课记录管理
